Image Class Reference

Detailed Description

A bitmap resource, stored in GPU memory. Use loadFromFile(const string& filepath) to load a image from a file. Helper class for the Vulkan resources VkImage, VkImageView and VkSampler

#include <image.h>

Creates a Image resource and store the image data in GPU memory. Use loadFromFile(const string& filepath) to load a image from a file.

Parameters
device: the GPU where the image will be stored
name: resource name. Informational only.
width: width in pixels
height: height in pixels
imageSize: data size in bytes
data: the source bitmap stored in CPU memory
format: data format of the bitmap (https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkFormat.html)
tiling: image tiling arrangment (https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkImageTiling.html)
samplerAddressMode: texture sampler mode (https://registry.khronos.org/vulkan/specs/1.3-extensions/man/html/VkSamplerAddressMode.html)
noMipmaps: do not generate mipmaps
